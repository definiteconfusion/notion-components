# Button

## Styles

- Light
- Dark
- Affirmative

## Functions

- mode: controls styles
- onClick: controls onClick functionality
- className: allows for additional tooling with custom classes

## Syntax

   ```html
   <Button
        mode="light"
        onClick={() => {console.log('Hello World')}}
    >
      Hello World, This is Light Mode
    </Button>
   ```
