# Link

## Styles

- Light
- Dark
- Affirmative

## Functions

- mode: controls styles
- color: provides manual color over-ride
- destination: sets the link's web destination
- className: allows for additional tooling with custom classes

## Syntax

   ```html
   <Link 
    destination="https://jakerase.dev"
    mode="affirmative"
    >
      Hello World
    </Link>
   ```
